# TROJI

**Troji** is a web application for tracking victories between members of a group. Users earn trophies in the form of emojis — hence the name — and can compare progress, celebrate wins, and compete socially with friends. The app is built around the idea that any shared achievement deserves recognition, with a lightweight emoji-based trophy system that keeps things fun and personal.

## Tech Stack

- **Backend**: .NET 10 GraphQL API (HotChocolate), PostgreSQL via EF Core, Clerk JWT authentication
- **Frontend**: React 19, React Router 7 (framework mode), Relay for GraphQL data fetching, Clerk for auth

## Status

The project is mid-rewrite. Core infrastructure, backend, and frontend architecture are complete. Feature pages (groups, games, invites, trophy requests) are currently in progress.
